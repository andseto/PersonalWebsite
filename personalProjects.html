Virtual Painter

Virtual Painter is a C++ program that uses a marker(s) that the camera looks for and draws the corresponding color. Use of code to enable the camera and an understanding how different colors have their own black and white color codes make this project possible.

This program uses the idea that each color has different characteristics for blacks and whites. For example, blue might need more black to become fully black. While red might need less black. This can all be recorded by color code within the program. When the color code is matched in the camera, it will draw circles of the corresponding color.
